# Instrumental Variables in Stata: Returns to Education: 
Esta carpeta contiene un proyecto de econometría aplicada implementado en Stata, centrado en la estimación de los retornos a la educación mediante MCO y Variables Instrumentales.

## Proyecto
- **Retornos a la Educación**
- Output: logaritmo del ingreso mensual
- Tratamiento: años de educación
- Instrumentos: educación de los padres

## Métodos
- MCO
- Variables Instrumentales (MC2E)
- Diagnóstico de instrumentos débiles
- Hausman test
- Sargan overidentification test

## Datos
- EMOVI 2015 (México)
- Submuestra de población adulta

## Archivos
- `Codigo_proyecto_econometria_aplicada.do`: archivo .do principal de Stata
- `Proyecto_econometria.pdf`: informe final
