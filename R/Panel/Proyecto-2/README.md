# Panel Data Models (Pooled, RE, FE)
Esta carpeta contiene un proyecto de datos de panel aplicados que utiliza estimadores de MCO agrupados, efectos aleatorios y efectos fijos.

## Proyecto
- **Empleo y Aprobación Presidencial**
- Output: aprobación del presidente (binario)
- Tratamiento: situación laboral (FT o PT)

## Métodos
- Pooled OLS
- Random Effects (RE)
- Fixed Effects (FE)
- Robust and clustered standard errors

## Datos
- Datos de encuestas de panel (2010, 2012, 2014)
- Observaciones a nivel individual

- `Proyecto_2.pdf`: Informe final compilado
