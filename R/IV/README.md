# Variables Instrumentales (IV)

Esta carpeta contiene ejercicios de econometría aplicada con Variables Instrumentales.

## Proyecto
- **Retornos de la Educación** (IV)
- Instrumento: proximidad a una universidad (nearc4)
- Output: logaritmo de los salarios

## Métodos
- OLS (línea base)
- IV (MC2E)
- Random Forest para la selección de controles
- Instrumental Causal Forest

## Archivos
- `Retornos_a_la_educ_AE&DS_script`: análisis principal y réplica
- `Proyecto_de_Ciencia_de_Datos.pdf`: informe final (resultado compilado)
