# Differences-in-Differences & Propensity Score Matching (DID–PSM)
Esta carpeta contiene un proyecto de microeconometría aplicada que combina métodos de DID, Event Studies y PSM.

## Proyecto
- **Efecto de jugar sin espectadores en la ventaja de local**
- Contexto: Cierre de estadios por COVID-19
- Caso práctico: Liga MX

## Métodos
- Diferencias en Diferencias (DID)
- Estudio de eventos
- Emparejamiento por Puntaje de Propensión (PSM)
- Errores estándar robustos y agrupados

## Datos
- Datos a nivel de partido de football-data.co.uk
- Cuotas de apuestas transformadas en probabilidades implícitas

## Archivos
- `Efecto_del_publico_en_los_estadios.Rmd`: código principal del análisis
- `Trabajo_final.pdf`: informe final

## Nota
- Los datos no están incluidos en el repositorio.
